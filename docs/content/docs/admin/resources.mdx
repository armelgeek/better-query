Admin resources extend Better Query resources with UI-specific configuration. This guide covers everything you need to know about creating and managing admin resources.

## What is an Admin Resource?

An admin resource is a Better Query resource enhanced with UI metadata. It defines:

- How data should be displayed (list, detail, forms)
- Which fields to show in different views
- How fields should be rendered and validated
- User interface elements (icons, labels, descriptions)

## Creating a Resource

Use `createAdminResource()` to create an admin resource:

```typescript
import { createAdminResource } from "better-admin";

const productAdmin = createAdminResource({
  name: "product",
  label: "Product",
  labelPlural: "Products",
  icon: "ðŸ“¦",
  description: "Manage your products",
  list: {
    fields: ["name", "price", "status"],
  },
  fieldMetadata: {
    name: { label: "Product Name", inputType: "text" },
    price: { label: "Price", inputType: "number" },
  },
});
```

### Create Form Configuration

The create form is used to create new resources:

```typescript
create: {
  // Fields to include in the form
  fields: ["name", "email", "password", "role", "status"],
  
  // Layout mode
  layout: "default", // "default" | "tabs" | "sections"
  
  // Default values
  defaultValues: {
    role: "user",
    status: "active",
  },
  
  // Sections (when layout is "sections")
  sections: [
    {
      title: "Basic Information",
      fields: ["name", "email"],
    },
    {
      title: "Account Settings",
      fields: ["password", "role", "status"],
    },
  ],
  
  // Submit button text
  submitLabel: "Create Product",
  
  // Success message
  successMessage: "Product created successfully!",
  
  // Redirect after create
  redirectAfterCreate: true,
  redirectPath: "/admin/products/:id", // :id will be replaced with new resource id
}
```

### Edit Form Configuration

```typescript
edit: {

  deleteButtonLabel: "Delete Product",

```

## Field Metadata

Field metadata defines how each field should be displayed and edited. See the [Configuration](/docs/admin/configuration) guide for complete field metadata options.

## Resource Relationships

Define relationships between resources:

```typescript
createAdminResource({
  name: "product",
  fieldMetadata: {
    categoryId: {
      label: "Category",
      inputType: "select",
      reference: "category",
      referenceField: "name",
    },
    tags: {
      label: "Tags",
      inputType: "multiselect",
      reference: "tag",
      referenceField: "name",
    },
  },
})
```

## Virtual Fields

Add computed fields that don't exist in the database:

```typescript
createAdminResource({
  name: "product",
  list: {
    fields: ["name", "price", "discountPrice", "savings"],
  },
  fieldMetadata: {
    savings: {
      label: "Savings",
      virtual: true,
      compute: (record) => record.price - record.discountPrice,
      formatter: (value) => `$${value.toFixed(2)}`,
    },
  },
})
```

## Access Control

Control access to resources and operations:

```typescript
createAdminResource({
  name: "product",
  permissions: {
    list: (ctx) => ctx.user?.role === "admin",
    create: (ctx) => ctx.user?.role === "admin",
    read: (ctx) => true, // Everyone can view
    update: (ctx) => ctx.user?.role === "admin",
    delete: (ctx) => ctx.user?.role === "admin",
  },
})
```

## Custom Actions

Add custom actions to resources:

```typescript

})
```

## Validation

Add custom validation rules:

```typescript
fieldMetadata: {
  email: {
    label: "Email",
    inputType: "email",
    validate: (value) => {
      if (!value.includes("@")) {
        return "Invalid email address";
      }
      return true;
    },
  },
  price: {
    label: "Price",
    inputType: "number",
    validate: (value, formData) => {
      if (formData.discountPrice && value < formData.discountPrice) {
        return "Price must be greater than discount price";
      }
      return true;
    },
  },
}
```

## Formatters and Parsers

Transform values for display and editing:

```typescript
fieldMetadata: {
  price: {
    label: "Price",
    inputType: "number",
    // Format for display
    formatter: (value) => `$${value.toFixed(2)}`,
    // Parse for editing
    parser: (value) => parseFloat(value.replace(/[^0-9.]/g, "")),
  },
  tags: {
    label: "Tags",
    inputType: "multiselect",
    formatter: (value) => value.join(", "),
    parser: (value) => value.split(",").map(v => v.trim()),
  },
}
```

## Examples

### E-commerce Product Resource

```typescript
createAdminResource({
  name: "product",
  label: "Product",
  labelPlural: "Products",
  icon: "ðŸ“¦",
  list: {
    fields: ["name", "price", "stock", "status"],
    defaultSort: "createdAt",
    defaultSortOrder: "desc",
    searchFields: ["name", "sku"],
  },
  fieldMetadata: {
    name: {
      label: "Product Name",
      inputType: "text",
      required: true,
    },
    sku: {
      label: "SKU",
      inputType: "text",
      required: true,
    },
    price: {
      label: "Price",
      inputType: "number",
      min: 0,
      formatter: (value) => `$${value.toFixed(2)}`,
    },
    stock: {
      label: "Stock",
      inputType: "number",
      min: 0,
    },
    status: {
      label: "Status",
      inputType: "select",
      options: [
        { label: "Draft", value: "draft" },
        { label: "Active", value: "active" },
        { label: "Out of Stock", value: "out_of_stock" },
      ],
    },
  },
})
```

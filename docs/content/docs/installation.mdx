### 1. Install Dependencies

Better Query requires `kysely` for database operations and `zod` for schema validation.

```bash
npm install better-query kysely zod
```

### 2. Initialize the Engine

Create a `lib/query.ts` file to initialize the engine with your database.

```typescript
import { betterQuery } from "better-query";
import { Kysely, SqliteDialect } from "kysely";
import Database from "better-sqlite3";

// Your Kysely DB instance
const db = new Kysely<any>({
  dialect: new SqliteDialect({
    database: new Database("db.sqlite"),
  }),
});

export const { handler, api } = betterQuery({
  database: db,
  resources: [
    // We will define resources in the next section
  ],
});
```

### 3. Mount the API

Better Query is framework-agnostic. Here is how to mount it:

#### **Hono**
```typescript
import { Hono } from "hono";
import { handler } from "./lib/query";

const app = new Hono();

// Match all query routes
app.all("/api/query/*", (ctx) => handler(ctx.req.raw));

export default app;
```

#### **Next.js** (App Router)
```typescript
// app/api/query/[...slug]/route.ts
import { handler } from "@/lib/query";

export const GET = (req: Request) => handler(req);
export const POST = (req: Request) => handler(req);
export const PATCH = (req: Request) => handler(req);
export const DELETE = (req: Request) => handler(req);
```

#### **Express**
```typescript
import express from "express";
import { handler } from "./lib/query";

const app = express();
app.use(express.json());

app.all("/api/query/*", async (req, res) => {
  // Convert Node request to Web Request
  const webReq = new Request(`http://${req.headers.host}${req.url}`, {
    method: req.method,
    headers: req.headers as any,
    body: req.method !== "GET" ? JSON.stringify(req.body) : undefined,
  });
  
  const response = await handler(webReq);
  const data = await response.json();
  res.status(response.status).json(data);
});
```

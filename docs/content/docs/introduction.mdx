Better Query is a high-performance, type-safe CRUD engine designed to automate the creation of REST APIs while providing advanced features required by modern enterprise applications.

### Core Philosophy

Unlike traditional CRUD generators, Better Query is:

- **Framework Agnostic**: Works with Hono, Express, Next.js, or any Node.js environment.
- **Enterprise Ready**: Built-in support for Multi-Tenancy, Field-Level Security, and Custom Business Actions.
- **Type-Safe**: End-to-end type safety from your database schema to your client code.
- **Plugin-Driven**: Easily extendable with Audit, Webhooks, Caching, and History tracking.

### Key Features

- **🛡️ Multi-Tenancy**: Automatic data isolation via `tenantId`.
- **⚡ Custom Actions**: Execute complex business logic at the resource level (e.g., `/project/:id/archive`).
- **🔒 Field Security**: Fine-grained control over field visibility and mutability.
- **🔄 Relational Aggregations**: Automatic subqueries for counts, sums, and averages across relationships.
- **🚀 Universal Client**: A single, type-safe client to consume your API in any environment.

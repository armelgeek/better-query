# List

The `List` component provides a container for displaying lists of records. It integrates with better-query for data fetching and provides pagination, filtering, and sorting capabilities.

## Usage

```tsx
import { List, DataTable } from '@/components/admin';

export const PostList = () => (
  <List resource="posts">
    <DataTable>
      <DataTable.Col source="title" />
      <DataTable.Col source="author" />
      <DataTable.Col source="created_at" />
    </DataTable>
  </List>
);
```

## Props

| Prop | Type | Description |
|------|------|-------------|
| `resource` | `string` | Resource name for API calls |
| `title` | `string` | Page title override |
| `children` | `ReactNode` | List content (typically DataTable) |
| `actions` | `ReactNode` | Custom action buttons |
| `filters` | `FilterConfig[]` | Available filters |

## Examples

### Basic List

```tsx
import { List, DataTable } from '@/components/admin';

export const PostList = () => (
  <List resource="posts">
    <DataTable>
      <DataTable.Col source="id" />
      <DataTable.Col source="title" />
      <DataTable.Col source="status" />
      <DataTable.Col source="created_at" />
    </DataTable>
  </List>
);
```

### With Custom Actions

```tsx
import { List, DataTable, CreateButton } from '@/components/admin';

export const PostList = () => (
  <List 
    resource="posts"
    actions={
      <div className="flex gap-2">
        <CreateButton />
        <Button variant="outline">Export</Button>
      </div>
    }
  >
    <DataTable>
      <DataTable.Col source="title" />
      <DataTable.Col source="author" />
    </DataTable>
  </List>
);
```

### With Filters

```tsx
import { List, DataTable } from '@/components/admin';

const filters = [
  { source: 'status', type: 'select', choices: ['draft', 'published'] },
  { source: 'author', type: 'text' },
];

export const PostList = () => (
  <List resource="posts" filters={filters}>
    <DataTable>
      <DataTable.Col source="title" />
      <DataTable.Col source="status" />
      <DataTable.Col source="author" />
    </DataTable>
  </List>
);
```

### With Custom Title

```tsx
import { List, DataTable } from '@/components/admin';

export const PostList = () => (
  <List resource="posts" title="All Blog Posts">
    <DataTable>
      <DataTable.Col source="title" />
      <DataTable.Col source="author" />
    </DataTable>
  </List>
);
```

## Data Integration

The List component automatically:
- Fetches data using better-query
- Handles pagination
- Manages loading and error states
- Provides data to child components

```tsx
// Data is automatically available to child components
export const PostList = () => (
  <List resource="posts">
    <DataTable>
      {/* DataTable receives the fetched data */}
      <DataTable.Col source="title" />
    </DataTable>
  </List>
);
```

## Installation

```bash
npx better-admin add list
```

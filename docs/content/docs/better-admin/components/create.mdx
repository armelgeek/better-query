# Create

The `Create` component provides a container for creating new records. It handles form submission, validation, and success/error states.

## Usage

```tsx
import { Create, SimpleForm, TextInput } from '@/components/admin';

export const PostCreate = () => (
  <Create resource="posts">
    <SimpleForm>
      <TextInput source="title" required />
      <TextInput source="content" multiline />
      <SelectInput source="status" choices={['draft', 'published']} />
    </SimpleForm>
  </Create>
);
```

## Props

| Prop | Type | Description |
|------|------|-------------|
| `resource` | `string` | Resource name for API calls |
| `title` | `string` | Page title override |
| `children` | `ReactNode` | Form content |
| `redirect` | `string \| false` | Redirect path after successful creation |
| `mutationOptions` | `object` | Custom mutation options |

## Examples

### Basic Create Form

```tsx
import { Create, SimpleForm, TextInput, SelectInput } from '@/components/admin';

export const PostCreate = () => (
  <Create resource="posts">
    <SimpleForm>
      <TextInput source="title" required />
      <TextInput source="content" multiline rows={5} />
      <SelectInput 
        source="status" 
        choices={[
          { id: 'draft', name: 'Draft' },
          { id: 'published', name: 'Published' }
        ]} 
      />
    </SimpleForm>
  </Create>
);
```

### With Custom Title

```tsx
import { Create, SimpleForm, TextInput } from '@/components/admin';

export const PostCreate = () => (
  <Create resource="posts" title="Create New Blog Post">
    <SimpleForm>
      <TextInput source="title" />
      <TextInput source="content" multiline />
    </SimpleForm>
  </Create>
);
```

### With Custom Redirect

```tsx

  <Create resource="posts" redirect="/posts">
    <SimpleForm>
      <TextInput source="title" />
      <TextInput source="content" multiline />
    </SimpleForm>
  </Create>
);
```

### No Redirect After Creation

```tsx
import { Create, SimpleForm, TextInput } from '@/components/admin';

export const PostCreate = () => (
  <Create resource="posts" redirect={false}>
    <SimpleForm>
      <TextInput source="title" />
      <TextInput source="content" multiline />
    </SimpleForm>
  </Create>

```

### With Validation

```tsx
import { Create, SimpleForm, TextInput } from '@/components/admin';
import { z } from 'zod';

const schema = z.object({
  title: z.string().min(1, 'Title is required'),
  content: z.string().min(10, 'Content must be at least 10 characters'),
});

export const PostCreate = () => (
  <Create resource="posts">
    <SimpleForm schema={schema}>
      <TextInput source="title" required />
      <TextInput source="content" multiline required />
    </SimpleForm>
  </Create>
);
```

### With Related Data

```tsx
import { Create, SimpleForm, TextInput, ReferenceInput, SelectInput } from '@/components/admin';

export const PostCreate = () => (
  <Create resource="posts">
    <SimpleForm>
      <TextInput source="title" required />
      <TextInput source="content" multiline />
      <ReferenceInput source="author_id" reference="users">
        <SelectInput optionText="name" />
      </ReferenceInput>
      <ReferenceInput source="category_id" reference="categories">
        <SelectInput optionText="name" />
      </ReferenceInput>
    </SimpleForm>
  </Create>
);
```

## Form Integration

The Create component automatically:
- Provides form context for input components
- Handles form submission via better-query
- Manages loading and error states
- Shows success notifications
- Handles redirects after successful creation

```tsx
// All input components receive form context automatically
export const PostCreate = () => (
  <Create resource="posts">
    <SimpleForm>
      {/* These inputs are automatically connected to the form */}
      <TextInput source="title" />
      <TextInput source="content" />
    </SimpleForm>
  </Create>
);
```

## Error Handling

The Create component automatically handles errors and displays them to the user:

```tsx
export const PostCreate = () => (
  <Create resource="posts">
    <SimpleForm>
      <TextInput source="title" />
      {/* Field-level errors are shown automatically */}
      <TextInput source="email" type="email" />
    </SimpleForm>
  </Create>
);
```

# Better Query v0.1.0

Initial release of Better Query - a type-safe CRUD API generator following the architecture patterns of better-auth.

## âœ¨ Features

- **Type-safe schemas** using Zod validation
- **Automatic CRUD endpoints** generation (Create, Read, Update, Delete, List)
- **Database adapters** for PostgreSQL, MySQL, SQLite with Kysely query builder
- **Framework integrations** for Next.js, Hono, and Express
- **Plugin system** for extensibility and custom business logic
- **Client libraries** for React and Vanilla JS with type safety
- **Better Auth integration** for seamless authentication and authorization
- **Role-based permissions** and organization-level access control
- **Pagination, filtering, and search** built-in support
- **Real-time updates** and optimistic updates support

## ðŸ”§ Core Components

### Schema Definition
```typescript
import { z } from 'zod';
import { createResource } from 'better-query';

const userSchema = z.object({
  id: z.string(),
  name: z.string(),
  email: z.string().email(),
  role: z.enum(['admin', 'user']),
});

export const userResource = createResource({
  name: 'user',
  schema: userSchema,
});
```

### Framework Integration
- **Next.js**: App Router and Pages Router support with universal handlers
- **Hono**: Native integration with Hono's handler system
- **Express**: Full compatibility with Express.js middleware

### Client Libraries
- **React**: Hooks and components for seamless frontend integration
- **Vanilla JS**: Framework-agnostic client for any JavaScript environment
- **Type Safety**: Full TypeScript support with inferred types from server

## ðŸš€ Getting Started

```bash
npm install better-query
```

```typescript
import { betterQuery, createResource } from 'better-query';
import { z } from 'zod';

const query = betterQuery({
  basePath: '/api/query',
  database: { provider: 'sqlite', url: 'data.db' },
  resources: [
    createResource({
      name: 'product',
      schema: z.object({
        id: z.string(),
        name: z.string(),
        price: z.number(),
      }),
    }),
  ],
});
```

# Better Query v0.1.0

Initial release of Better Query - a type-safe CRUD API generator.

## Features

- **Type-safe schemas** using Zod
- **Automatic CRUD endpoints** generation
- **Database adapters** for PostgreSQL, MySQL, SQLite
- **Plugin system** for extensibility
- **Client libraries** for React and Vanilla JS
- **Framework integrations** for Next.js, Hono, and Express

## Breaking Changes

This is the initial release, so there are no breaking changes from previous versions.

## Documentation

- Added comprehensive documentation for all features
- Added example projects for Next.js integration
- Added guides for database setup and plugin development
